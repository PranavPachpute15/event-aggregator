# Event Aggregator Platform 🚀

## 📌 Overview
A full-stack system that automatically collects, processes, and displays tech events like hackathons, workshops, and conferences.

## ⚙️ Tech Stack
- Backend: Flask (Python)
- Database: MySQL
- Scraping: BeautifulSoup
- API: REST

## 🔥 Features
- Automated event scraping
- Data aggregation from multiple sources
- Duplicate removal
- Category-based filtering
- REST API endpoints
- Clean architecture (routes, services, models)
